Practice — Part 1 of 2
Stroop practice

A color word will appear on screen. Click the button that matches the ink color — not what the word says.

Example:

BLUE

Correct answer: Red — because the ink is red.

Practice results are not recorded. Go as fast and accurately as you can.
Practice — Part 2 of 2
N-back practice

Letters appear one at a time. Each time, decide: does this letter match the one from N steps ago?

Example — 2-back:

You see: K

Was the letter shown 2 turns ago also K? Answer from memory only — no hints shown.

Warm-up: Each N-back block starts with a few silent loading letters — just watch them, no response needed. They seed your memory so the first real trial is always fair. N is either 2 or 3 and is always shown on screen. Practice results are not recorded.
